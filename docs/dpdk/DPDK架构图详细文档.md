### EAL初始化交互

```
应用程序
    │
    │ rte_eal_init()
    ▼
┌─────────────────┐
│  EAL初始化      │
└────────┬────────┘
         │
         ├─► eal_create_runtime_dir()
         │   └─► 创建 /var/run/dpdk
         │
         ├─► eal_parse_args()
         │   └─► 解析命令行参数
         │
         ├─► eal_hugepage_info_init()
         │   └─► 读取 /sys/kernel/mm/hugepages
         │
         ├─► eal_memalloc_init()
         │   └─► 初始化内存分配器
         │
         ├─► rte_eal_memory_init()
         │   ├─► 分配大页内存
         │   ├─► 创建内存段
         │   └─► 映射到用户空间
         │
         ├─► rte_eal_pci_init()

         │
```

### 内存分配流程

```
应用程序请求内存
    │
    ▼
┌──────────────┐
│ 本地缓存检查 │
└──────┬───────┘
       │
       ├─► 有对象 ──► 返回对象 (快速路径，~10 cycles)
       │
       └─► 无对象
              │
              ▼
       ┌──────────────┐
       │ 从Ring获取   │
       └──────┬───────┘
              │
              ├─► 批量获取 (如32个对象)
              │
              ▼
       ┌──────────────┐
       │ 填充本地缓存 │
       └──────┬───────┘
              │
              ▼
       返回一个对象给应用
```

### 内存释放流程

```
应用程序释放内存
    │
    ▼
┌──────────────┐
│ 本地缓存检查 │
└──────┬───────┘
       │
       ├─► 未满 ──► 放入本地缓存 (快速路径，~10 cycles)
       │
       └─► 已满
              │
              ▼
       ┌──────────────┐
       │ 批量刷新到Ring│
       └──────┬───────┘
              │
              ├─► 清空本地缓存
              │
              ▼
       ┌──────────────┐
       │ 放入新对象   │
       └──────────────┘
```
